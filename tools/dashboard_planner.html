Tippe eine Box an, um sie zu verschieben/skalieren, oder klicke ein einzelnes Element (Text, Button, Icon, Schalter, Slider) darin an, um es einzeln zu bearbeiten und frei zu platzieren. Doppelklick auf Text bearbeitet ihn direkt.
Box ziehen zum Verschieben (Rand/Hintergrund anfassen) · Element anklicken zum Auswählen und frei innerhalb der Box zu platzieren · blauer Punkt zum Skalieren · Doppelklick auf Text zum direkten Bearbeiten · Entf zum Zurücksetzen der Auswahl · Export liefert eine YAML-Planungsdatei zur manuellen Übernahme ins echte dashboard.yaml.

Layout-Export (YAML)

Enthält jede Box mit all ihren Elementen (Typ, Text, Position, Größe, Schriftgröße, Farbe). Planungsdatei, kein direkt flashbares ESPHome-YAML — schick sie zurück (Text einfügen oder Datei anhängen), dann übertrage ich die Änderungen ins echte dashboard.yaml, kompiliere und flashe per USB.
